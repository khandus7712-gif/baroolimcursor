# 바로올림 데모 영상 스크립트

## 전체 내레이션 스크립트 (2분 30초)

---

### [0:00-0:15] 오프닝

**화면:** 바로올림 로고 페이드인

**내레이션:**
```
안녕하세요, 바로올림입니다.
사진 1장과 간단한 메모만으로
모든 채널 콘텐츠를 자동 생성하는
AI 마케팅 콘텐츠 생성기입니다.
```

**텍스트 오버레이:** "바로올림 - AI 마케팅 콘텐츠 생성기"

---

### [0:15-0:30] 문제 제시

**화면:** 자영업자 일러스트/아이콘 또는 통계 그래프

**내레이션:**
```
자영업자들은 매일 인스타그램, 블로그,
Google My Business에 올릴 콘텐츠를
만드는 데 많은 시간을 소비합니다.

전문 마케터를 고용하기에는 비용이 부담스럽고,
직접 만들기에는 전문성이 부족합니다.
```

**텍스트 오버레이:** 
- "시간 부족"
- "비용 부담"
- "전문성 부족"

---

### [0:30-0:45] 해결책 소개

**화면:** 스튜디오 페이지 전체 보기

**내레이션:**
```
바로올림은 이 문제를 해결합니다.
3단계로 간단하게 콘텐츠를 생성할 수 있습니다.
```

**텍스트 오버레이:** "3단계로 간단하게"

---

### [0:45-1:00] 업종 선택

**화면:** 랜딩 페이지 → 업종 선택 (음식/식당 클릭)

**내레이션:**
```
첫 번째, 업종을 선택합니다.
음식점, 미용실, 소매점 등
업종별로 최적화된 콘텐츠가 생성됩니다.
```

**하이라이트:** 선택된 업종 강조

---

### [1:00-1:15] 스튜디오 접속 및 웹 검색 기능

**화면:** 스튜디오 페이지 → "기존 콘텐츠 조사" 체크박스 강조

**내레이션:**
```
스튜디오에 접속하면
간단한 입력만으로 콘텐츠를 생성할 수 있습니다.

특히 이 기능은 뤼튼과 같은 수준의
웹 검색 기반 조사 기능입니다.
관련 블로그 포스트를 자동으로 검색하여
더 풍부하고 트렌드에 맞는 콘텐츠를 생성합니다.
```

**하이라이트:** "기존 콘텐츠 조사 (NEW ✨)" 체크박스 펄스 효과

---

### [1:15-1:30] 이미지 업로드

**화면:** 이미지 드래그 앤 드롭 또는 클릭

**내레이션:**
```
두 번째, 사진을 업로드합니다.
AI가 이미지를 자동으로 분석하여
콘텐츠에 반영합니다.
```

**하이라이트:** 이미지 미리보기 표시

---

### [1:30-1:45] 메모 입력

**화면:** 메모 입력 필드에 텍스트 입력

**내레이션:**
```
세 번째, 간단한 메모를 입력합니다.
예를 들어, '오늘의 특선 메뉴: 아롱사태 전골'
이런 식으로 간단히 입력하면 됩니다.
```

**하이라이트:** 입력되는 텍스트 강조

---

### [1:45-2:00] 생성 버튼 클릭

**화면:** 생성 버튼 클릭 → 로딩 애니메이션

**내레이션:**
```
이제 생성 버튼을 누르면...
AI가 웹에서 관련 콘텐츠를 검색하고,
4개 플랫폼에 최적화된 콘텐츠를 생성합니다.
```

**하이라이트:** 로딩 스피너, "검색 중..." 표시

---

### [2:00-2:15] 결과 확인 - 멀티 플랫폼

**화면:** 생성된 콘텐츠 결과 화면

**내레이션:**
```
3초 만에 4개 플랫폼에 최적화된
콘텐츠가 자동으로 생성됩니다.

Instagram, Blog, Threads, Google My Business
각 플랫폼의 특성에 맞게 최적화되어 있습니다.
```

**하이라이트:** 각 플랫폼별 콘텐츠 순차 표시

**텍스트 오버레이:**
- "3초 만에 완성"
- "4개 플랫폼 동시 생성"
- "플랫폼별 최적화"

---

### [2:15-2:30] 차별화 포인트

**화면:** 비교표 또는 아이콘

**내레이션:**
```
뤼튼과 비교했을 때,
바로올림의 차별화 포인트는
멀티 플랫폼 지원과 업종별 특화입니다.

뤼튼은 블로그만 지원하지만,
우리는 4개 플랫폼을 동시에 지원합니다.
또한 업종별 특화 콘텐츠로
더 정확하고 효과적인 마케팅이 가능합니다.
```

**텍스트 오버레이:**
- "멀티 플랫폼"
- "업종 특화"
- "예약 발행"

---

### [2:30-2:45] 클로징

바로올림으로 마케팅 콘텐츠를

## 촬영 시 주의사항

### 타이밍
- 각 씬은 정확한 시간에 맞춰 촬영
- 내레이션과 화면 동작이 일치하도록
- 너무 빠르지 않게, 천천히 진행

### 강조 포인트
- "기존 콘텐츠 조사" 기능은 반드시 강조
- 멀티 플랫폼 지원 강조
- 업종별 특화 강조

### 화면 구성
- 마우스 커서가 명확하게 보이게
- 클릭하는 부분을 명확하게 표시
- 로딩 시간은 실제보다 약간 빠르게 편집 가능

## 내레이션 녹음 팁

### 목소리 톤
- 전문적이면서도 친근하게
- 분당 150-180단어 속도
- 명확한 발음
- 자연스러운 억양

### 강조할 단어
- **"3초 만에"** → 강조
- **"4개 플랫폼"** → 강조
- **"자동으로"** → 강조
- **"뤼튼과 달리"** → 강조
- **"업종별 특화"** → 강조

### 휴지
- 문장 사이에 자연스러운 휴지
- 중요한 메시지 전후에 약간의 휴지
- 너무 길지 않게 (0.5-1초)

## 편집 시 추가할 요소

- 핵심 메시지 강조

- 중요 부분 줌 인
- 클릭 부분 하이라이트
- 전환 효과 (페이드)

### 배경 음악
- 부드러운 BGM (볼륨 30-40%)
- 업템포 (오프닝)
- 미디엄 (본문)
- 다운템포 (클로징)
