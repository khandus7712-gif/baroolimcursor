# 바로올림 데모 영상 제작 가이드

투자자 피칭용 2-3분 데모 영상 제작 가이드입니다.

---

## 📹 영상 개요

**목표:** 바로올림의 핵심 기능을 2-3분 안에 효과적으로 보여주기  
**대상:** 투자자, 잠재 고객  
**톤:** 전문적이면서도 친근하게

---

## 🎬 영상 구성 (2분 30초)

### 0:00 - 0:15 (15초) | 오프닝
- **화면:** 바로올림 로고 + 슬로건
- **내레이션:** "사진 1장으로 모든 채널 콘텐츠를 자동 생성하는 바로올림입니다"
- **시각:** 로고 페이드인 → 메인 화면으로 전환

### 0:15 - 0:30 (15초) | 문제 제시
- **화면:** 자영업자의 고민을 시각화 (아이콘/일러스트)
- **내레이션:** "자영업자들은 매일 인스타, 블로그, GMB에 올릴 콘텐츠를 만드는 데 시간이 많이 걸립니다"
- **텍스트 오버레이:** "시간 부족", "비용 부담", "전문성 부족"

### 0:30 - 1:30 (60초) | 핵심 기능 시연
**가장 중요한 부분 - 천천히, 명확하게!**

#### 0:30 - 0:45 | 업종 선택
- **화면:** 랜딩 페이지 → 업종 선택 (음식/식당)
- **내레이션:** "업종을 선택하면 자동으로 맞춤 콘텐츠가 생성됩니다"
- **하이라이트:** 업종별 특화 강조

#### 0:45 - 1:00 | 스튜디오 접속
- **화면:** 스튜디오 페이지 전체 보기
- **내레이션:** "스튜디오에서 간단한 입력만으로 콘텐츠를 생성합니다"
- **하이라이트:** "기존 콘텐츠 조사 (NEW ✨)" 체크박스 강조

#### 1:00 - 1:15 | 이미지 업로드
- **화면:** 이미지 드래그 앤 드롭
- **내레이션:** "사진을 업로드하면 AI가 자동으로 분석합니다"
- **하이라이트:** 이미지 미리보기

#### 1:15 - 1:30 | 콘텐츠 생성
- **화면:** 메모 입력 → 생성 버튼 클릭 → 로딩 애니메이션
- **내레이션:** "간단한 메모만 입력하고 생성 버튼을 누르면..."
- **하이라이트:** "기존 콘텐츠 조사" 기능 작동 중 표시

### 1:30 - 2:00 (30초) | 결과 확인
- **화면:** 생성된 콘텐츠 결과 화면
- **내레이션:** "4개 플랫폼에 최적화된 콘텐츠가 자동으로 생성됩니다"
- **하이라이트:** 
  - Instagram 포스트
  - Blog 포스트
  - Threads 포스트
  - GMB 포스트
- **텍스트 오버레이:** "3초 만에 완성", "플랫폼별 최적화"

### 2:00 - 2:15 (15초) | 차별화 포인트
- **화면:** 비교표 또는 아이콘
- **내레이션:** "뤼튼과 달리 4개 플랫폼을 동시에 지원하고, 업종별 특화 콘텐츠를 제공합니다"
- **텍스트 오버레이:** "멀티 플랫폼", "업종 특화", "예약 발행"

### 2:15 - 2:30 (15초) | 클로징
- **화면:** 바로올림 로고 + CTA
- **내레이션:** "바로올림으로 마케팅 콘텐츠를 더 쉽고 빠르게 만들어보세요"
- **텍스트 오버레이:** "지금 시작하기" 또는 "데모 체험하기"

## 🎥 촬영 준비사항

### 필수 도구
1. **화면 녹화 소프트웨어**
   - Windows: OBS Studio (무료), Camtasia (유료)
   - Mac: QuickTime Player (무료), ScreenFlow (유료)
   - 온라인: Loom (무료, 웹 기반)

2. **마이크**
   - USB 마이크 또는 헤드셋 마이크
   - 내레이션 녹음용

3. **편집 소프트웨어**
   - 무료: DaVinci Resolve, Shotcut
   - 유료: Adobe Premiere Pro, Final Cut Pro

### 촬영 전 체크리스트
- [ ] 브라우저 창 크기 조정 (1920x1080 권장)
- [ ] 브라우저 확장 프로그램 비활성화 (깔끔한 화면)
- [ ] 데모용 계정 준비 (테스트 데이터)
- [ ] 네트워크 연결 확인 (빠른 로딩)
- [ ] 화면 밝기 조정 (너무 밝지 않게)
- [ ] 마우스 커서 크기 조정 (명확하게 보이게)

```

사진 1장과 간단한 메모만으로

AI 마케팅 콘텐츠 생성기입니다."
```

### Scene 2: 문제 제시
```
[화면: 자영업자 일러스트/아이콘]
[내레이션]
"자영업자들은 매일 인스타그램, 블로그,
Google My Business에 올릴 콘텐츠를

전문 마케터를 고용하기에는 비용이 부담스럽고,
직접 만들기에는 전문성이 부족합니다."
```

### Scene 3: 해결책 소개
```
[화면: 스튜디오 페이지]
[내레이션]
"바로올림은 이 문제를 해결합니다.
3단계로 간단하게 콘텐츠를 생성할 수 있습니다."
```

### Scene 4: 기능 시연
```
[화면: 업종 선택]
"첫 번째, 업종을 선택합니다.
음식점, 미용실, 소매점 등
업종별로 최적화된 콘텐츠가 생성됩니다."

[화면: 이미지 업로드]
"두 번째, 사진을 업로드합니다.
AI가 이미지를 자동으로 분석하여
콘텐츠에 반영합니다."

[화면: 메모 입력]
"세 번째, 간단한 메모를 입력합니다.
예를 들어, '오늘의 특선 메뉴: 아롱사태 전골'
이런 식으로 간단히 입력하면 됩니다."

[화면: "기존 콘텐츠 조사" 체크박스 강조]
"특히 이 기능은 뤼튼과 같은 수준의
웹 검색 기반 조사 기능입니다.
관련 블로그 포스트를 자동으로 검색하여
더 풍부하고 트렌드에 맞는 콘텐츠를 생성합니다."

[화면: 생성 버튼 클릭]
"이제 생성 버튼을 누르면..."
```

### Scene 5: 결과 확인
```
[화면: 생성된 콘텐츠 결과]
"3초 만에 4개 플랫폼에 최적화된
콘텐츠가 자동으로 생성됩니다.
Instagram, Blog, Threads, Google My Business
각 플랫폼의 특성에 맞게 최적화되어 있습니다."

[화면: 각 플랫폼별 콘텐츠 하이라이트]
"Instagram은 짧고 임팩트 있게,
블로그는 상세하고 SEO 최적화되게,
Threads는 대화형으로,
GMB는 검색 최적화되게 생성됩니다."
```

### Scene 6: 차별화 포인트
```
[화면: 비교표 또는 아이콘]

바로올림의 차별화 포인트는
멀티 플랫폼 지원과 업종별 특화입니다.
뤼튼은 블로그만 지원하지만,
우리는 4개 플랫폼을 동시에 지원합니다.
또한 업종별 특화 콘텐츠로
더 정확하고 효과적인 마케팅이 가능합니다."
```

### Scene 7: 클로징

## 🎨 시각적 효과 가이드

### 화면 전환
- **페이드:** 부드러운 전환 (0.3초)
- **줌 인/아웃:** 중요 부분 강조
- **하이라이트:** 마우스 커서로 클릭 부분 강조

### 텍스트 오버레이
- **폰트:** 깔끔한 산세리프 (Noto Sans, Pretendard)
- **크기:** 화면의 10-15% 정도
- **위치:** 상단 또는 하단
- **색상:** 브랜드 컬러 (주황색 계열)

### 애니메이션
- **로딩:** 부드러운 스피너
- **결과 표시:** 페이드인 효과
- **하이라이트:** 펄스 효과

## ✂️ 편집 가이드

### 편집 순서
1. **컷 편집:** 불필요한 부분 제거
2. **전환 효과:** 페이드, 줌 등 추가
3. **텍스트 오버레이:** 핵심 메시지 추가
4. **내레이션 추가:** 녹음된 음성 삽입
5. **배경 음악:** 부드러운 BGM 추가 (선택)
6. **색상 보정:** 화면 밝기/대비 조정
7. **최종 확인:** 전체 재생 확인

### 권장 편집 소프트웨어
- **무료:** DaVinci Resolve, Shotcut, OpenShot
- **유료:** Adobe Premiere Pro, Final Cut Pro, Camtasia
