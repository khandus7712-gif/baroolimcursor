# 🎯 최종 해결 가이드

## 현재 상황
- 서버가 자꾸 중지됨
- OAuth 설정은 되어있지만 Client Secret 문제

## ✅ 완벽한 해결 순서

### 1단계: 서버 안정화
```bash
npm run dev
```

### 2단계: Google Client Secret 재발급

**이것이 가장 중요합니다!**

1. https://console.cloud.google.com/ 접속
2. API 및 서비스 → 사용자 인증 정보
3. OAuth 2.0 클라이언트 ID 클릭
4. "추가 보안 비밀" 또는 "보안 비밀 재설정" 클릭
5. **새로운 보안 비밀을 복사**
6. `.env` 파일과 `.env.local` 파일 모두 업데이트:
   ```
   GOOGLE_CLIENT_SECRET=새로운_시크릿
   ```

### 3단계: 서버 재시작

```bash
Ctrl+C
npm run dev
```

### 4단계: 테스트

`http://localhost:3000/login`

---

## 🔧 또는 간단한 방법

이메일 로그인으로 먼저 테스트:
1. 로그인 페이지에서 이메일 입력
2. "이메일로 시작하기" 클릭
3. 작동하는지 확인

이메일 로그인이 작동하면 최소한 서버와 데이터베이스는 정상입니다!
