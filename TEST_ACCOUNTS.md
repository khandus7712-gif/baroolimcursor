## 🚀 사용 방법

### 1. 로그인
1. `/login` 페이지 접속
2. 이메일과 비밀번호 입력
3. "이메일로 시작하기" 버튼 클릭

### 2. 결제 테스트
1. 로그인 후 `/payment` 페이지 접속
2. 원하는 플랜 선택
3. 토스페이먼츠 테스트 카드로 결제 진행
