# 웹 검색 기능 활성화 체크리스트 ✅

## 1단계: Tavily API 키 발급 (필수)

- [ ] [Tavily 웹사이트](https://tavily.com) 접속
- [ ] 회원가입 또는 로그인
- [ ] 대시보드에서 API 키 발급
  - 무료 티어: 월 1,000회 검색 제공
  - 유료 플랜도 선택 가능

## 2단계: 환경 변수 설정

- [ ] 프로젝트 루트의 `.env` 파일 열기 (없으면 생성)
- [ ] 다음 줄 추가:

```env
TAVILY_API_KEY=여기에_발급받은_API_키_입력
```

**예시:**
```env
TAVILY_API_KEY=tvly-xxxxxxxxxxxxxxxxxxxxx
```

## 3단계: 개발 서버 재시작

- [ ] 개발 서버가 실행 중이면 중지 (Ctrl+C)
- [ ] 다시 시작: `npm run dev`
- [ ] 환경 변수 변경 후에는 반드시 재시작 필요!

## 4단계: 기능 테스트

- [ ] 브라우저에서 `/studio` 페이지 접속
- [ ] 메모와 키워드 입력
- [ ] "기존 콘텐츠 조사 (NEW ✨)" 체크박스가 체크되어 있는지 확인
- [ ] 콘텐츠 생성 버튼 클릭
- [ ] 콘텐츠가 생성되는지 확인
  - 검색 기능이 작동하면 약간 더 시간이 걸릴 수 있음 (2-5초 추가)
  - 더 풍부하고 트렌드에 맞는 콘텐츠가 생성되어야 함

## 5단계: 배포 환경 설정 (Vercel 등)

- [ ] 배포 플랫폼(Vercel, Railway 등)의 환경 변수 설정 페이지 접속
- [ ] `TAVILY_API_KEY` 환경 변수 추가
- [ ] 재배포 또는 환경 변수 업데이트 후 재배포

## 문제 해결

### API 키가 없어도 작동하나요?
✅ 네, API 키가 없으면 웹 검색 없이 기존 방식으로 작동합니다.
- 콘솔에 경고 메시지가 표시되지만 기능은 정상 작동합니다.

### 검색이 안 되나요?
1. `.env` 파일에 `TAVILY_API_KEY`가 올바르게 설정되었는지 확인
2. 개발 서버를 재시작했는지 확인
3. 브라우저 콘솔(F12)에서 에러 메시지 확인
4. API 키가 유효한지 Tavily 대시보드에서 확인

### 검색이 너무 느려요?
- 웹 검색은 약 2-5초 추가 시간이 소요됩니다
- 필요시 "기존 콘텐츠 조사" 체크박스를 해제하여 비활성화 가능
