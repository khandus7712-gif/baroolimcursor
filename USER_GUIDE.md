# 바로올림 사용자 가이드 📘

## 소개

바로올림은 AI를 활용한 마케팅 콘텐츠 자동 생성 서비스입니다. 이미지와 간단한 메모만으로 Instagram, Blog, Threads, Google My Business 등 여러 플랫폼에 최적화된 콘텐츠를 자동으로 생성합니다.

---

## 시작하기

### 1. 업종 선택

메인 페이지에서 업종을 선택하세요:
- **음식/식당**: 레스토랑, 카페, 배달 음식 등
- **뷰티/미용**: 미용실, 네일샵, 피부관리 등
- **소매/유통**: 의류, 액세서리, 생활용품 등

### 2. 스튜디오로 이동

"시작하기" 버튼을 클릭하면 콘텐츠 생성 스튜디오로 이동합니다.

---

## 콘텐츠 생성하기

### 입력 영역 (좌측)

#### 이미지 업로드 (선택사항)
- 클릭하거나 드래그 & 드롭으로 이미지 업로드
- 지원 형식: JPG, PNG, GIF, WEBP
- 최대 크기: 10MB
- 이미지가 있으면 AI가 이미지를 분석하여 콘텐츠에 반영합니다

#### 메모 입력
```
예시:
- 오늘의 특별 메뉴는 시그니처 파스타입니다
- 런치 세트 20% 할인 이벤트 진행 중
- 신선한 재료로 만든 홈메이드 파스타
```

#### 키워드 입력 (선택사항)
쉼표로 구분하여 입력:
```
예시: 파스타, 이탈리안, 런치, 할인
```

---

### 설정 영역 (우측)

#### 1. 업종 선택
- 음식/식당
- 뷰티/미용
- 소매/유통

#### 2. 플랫폼 선택
하나를 선택하세요:
- **Instagram**: 짧고 감성적인 콘텐츠
- **Blog**: 길고 자세한 콘텐츠
- **Threads**: 짧고 캐주얼한 콘텐츠
- **Google My Business**: 정보성 콘텐츠

#### 3. 플러그인 선택 (복수 선택 가능)
- **리뷰 답변**: 고객 리뷰에 답변하는 톤
- **광고 카피**: 판매 촉진 문구 추가
- **예약 CTA**: 예약 유도 문구 추가
- **해시태그**: 관련 해시태그 자동 생성 (Instagram용)

#### 4. 추가 설정 (선택사항)
- **브랜드 이름**: 콘텐츠에 브랜드 이름 포함
- **지역**: 위치 정보 포함
- **링크**: 예약 또는 주문 링크 추가
- **톤 힌트**: 원하는 말투나 스타일 (예: "친근하게", "전문적으로")

---

### 3. 생성 버튼 클릭

"생성하기" 버튼을 클릭하면 AI가 콘텐츠를 생성합니다.
- 생성 시간: 약 5-10초
- 실패 시 오류 메시지가 표시됩니다

---

## 결과 확인하기

### 생성된 콘텐츠

생성된 콘텐츠가 하단에 표시됩니다:
- 본문 텍스트
- 해시태그 (선택한 경우)
- 경고 메시지 (금칙어 대체, 필수 문구 삽입 등)

### 복사하기

"복사" 버튼을 클릭하면 클립보드에 복사됩니다. 원하는 플랫폼에 붙여넣기하세요.

## 사용 팁

### 좋은 메모 작성법

✅ **좋은 예:**
```
오늘의 신메뉴 시그니처 파스타 출시
신선한 바질과 토마토를 사용한 크림 파스타
런치 세트 12,000원 (음료 포함)
오후 2시까지 한정
```

❌ **나쁜 예:**
```
파스타
```

### 이미지 활용

- **제품 사진**: 음식, 상품, 서비스 사진
- **매장 사진**: 인테리어, 분위기
- **이벤트 사진**: 할인, 프로모션 안내

### 플랫폼별 특징

#### Instagram
- 짧고 감성적
- 해시태그 많이 사용
- 이모티콘 활용

#### Blog
- 길고 자세한 설명
- SEO 최적화
- 정보성 콘텐츠

#### Threads
- 짧고 캐주얼
- 대화형 톤
- 간결한 문장

#### Google My Business
- 정보 중심
- 위치, 영업시간 등 포함
- 전문적인 톤

## 자주 묻는 질문 (FAQ)

### Q: 이미지 없이도 콘텐츠를 생성할 수 있나요?
A: 네, 메모만으로도 생성 가능합니다. 이미지가 있으면 더 구체적인 콘텐츠를 생성합니다.

### Q: 생성된 콘텐츠를 수정할 수 있나요?
A: 생성 후 복사하여 원하는 대로 수정할 수 있습니다.

### Q: 한 번에 여러 플랫폼의 콘텐츠를 생성할 수 있나요?
A: 현재는 한 번에 하나의 플랫폼만 선택 가능합니다. 각 플랫폼별로 생성하세요.

### Q: 생성 비용이 있나요?
A: 무료 플랜은 하루 60회까지 가능합니다.

### Q: 콘텐츠가 마음에 들지 않으면?
A: "생성하기"를 다시 클릭하면 새로운 콘텐츠가 생성됩니다.

### Q: 금칙어는 무엇인가요?
A: 사용하면 안 되는 단어들입니다. 자동으로 대체되며 경고 메시지가 표시됩니다.

## 예제

### 음식/식당 - Instagram

**입력:**
- 이미지: 파스타 사진
- 메모: "오늘의 신메뉴 시그니처 파스타. 신선한 바질과 토마토 크림 소스"
- 키워드: "파스타, 이탈리안, 신메뉴"
- 플랫폼: Instagram
- 플러그인: 해시태그

**출력:**
```
🍝 오늘의 신메뉴가 찾아왔어요!

신선한 바질과 토마토가 만나 
특별한 크림 소스로 탄생한 
시그니처 파스타 🌿🍅

한입 베어 물면 입안 가득 퍼지는 
진한 풍미를 경험해보세요!

#파스타 #이탈리안 #신메뉴 #맛집 #오늘의메뉴 
#먹스타그램 #foodstagram #크림파스타
```

## 문제 해결

### 생성 실패

**오류:** "콘텐츠 생성에 실패했습니다"

**해결:**
1. 인터넷 연결 확인
2. 메모 내용 확인 (너무 짧거나 비어있지 않은지)
3. 잠시 후 다시 시도

### 이미지 업로드 실패

**오류:** "이미지 업로드에 실패했습니다"

**해결:**
1. 파일 형식 확인 (JPG, PNG, GIF, WEBP만 가능)
2. 파일 크기 확인 (10MB 이하)
3. 이미지 없이 메모만으로 생성 시도
