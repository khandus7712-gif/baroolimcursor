# 웹 검색 기능 설정 가이드

바로올림의 웹 검색 기능은 관련 블로그 포스트와 콘텐츠를 자동으로 검색하여, 더 풍부하고 트렌드에 맞는 콘텐츠를 생성할 수 있도록 도와줍니다.

## 기능 개요

웹 검색 기능이 활성화되면:
1. 사용자가 입력한 메모와 키워드를 기반으로 관련 콘텐츠를 자동 검색
2. 검색된 블로그 포스트와 기사를 분석하여 프롬프트에 포함
3. AI가 검색 결과를 참고하여 더 정확하고 트렌드에 맞는 콘텐츠 생성
4. 원본 콘텐츠를 복사하지 않고, 참고하여 독창적인 콘텐츠 생성

## Tavily API 설정

### 1. Tavily API 키 발급

1. [Tavily 웹사이트](https://tavily.com)에 접속
2. 회원가입 또는 로그인
3. 대시보드에서 API 키 발급 (무료 티어 제공)

### 2. 환경 변수 설정

`.env` 파일에 다음 변수를 추가:

```env
TAVILY_API_KEY=your-tavily-api-key-here
```

### 3. 기능 사용

스튜디오 페이지에서 콘텐츠 생성 시:
- "기존 콘텐츠 조사 (NEW ✨)" 체크박스가 기본적으로 활성화되어 있습니다
- 체크박스를 해제하면 웹 검색 없이 기존 방식으로 콘텐츠를 생성합니다
- 체크박스를 활성화하면 관련 콘텐츠를 검색하여 더 풍부한 결과를 얻을 수 있습니다

## 주의사항

- Tavily API 키가 설정되지 않은 경우, 웹 검색 기능은 자동으로 비활성화되고 기존 방식으로 동작합니다
- 웹 검색은 약간의 추가 시간이 소요될 수 있습니다 (약 2-5초)
- 검색 결과는 참고용으로만 사용되며, 원본 콘텐츠를 복사하지 않습니다

## 무료 티어 제한

Tavily의 무료 티어는:
- 월 1,000회 검색 제한
- 기본 검색 깊이 제공

더 많은 검색이 필요한 경우 유료 플랜을 고려하세요.

## 대안 API

Tavily API 외에도 다음 API를 사용할 수 있습니다 (코드 수정 필요):
- Serper API
- Google Custom Search API
- Bing Search API

현재는 Tavily API만 지원하며, 다른 API를 사용하려면 `lib/webSearch.ts` 파일을 수정해야 합니다.
