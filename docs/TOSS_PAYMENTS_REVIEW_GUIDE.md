   - 4개 플랫폼 지원

   - 4개 플랫폼 지원
   - 예약 발행 무제한
   - 가격: 29,900원/월

4. **Pro 플랜 (월간 구독)**
   - 하루 10개 생성 (월 300개)

5. **Enterprise 플랜 (월간 구독)**

   - 1:1 전담 지원

**서비스 특징:**
- 업종별 맞춤형 콘텐츠 생성
- 이미지 분석 및 자동 해시태그 생성

### 4. 결제경로 파일을 PPT로 제작해서 첨부해주세요.

**결제경로 파일 제작 가이드:**

결제경로 파일은 사용자가 결제를 진행하는 전체 과정을 스크린샷으로 보여주는 PPT 파일입니다.

**필수 포함 요소:**

1. **메인 페이지 접속**
   - 스크린샷: https://baroolim.com 메인 페이지
   - 설명: "바로올림 메인 페이지 접속"

2. **요금제 페이지 이동**
   - 스크린샷: https://baroolim.com/pricing 페이지
   - 설명: "요금제 선택 페이지에서 원하는 플랜 선택"

3. **결제 페이지 이동**
   - 스크린샷: https://baroolim.com/payment?plan=BASIC (예시)
   - 설명: "결제 페이지로 이동하여 결제 정보 확인"

4. **토스페이먼츠 결제창**
   - 스크린샷: 토스페이먼츠 결제창 화면
   - 설명: "토스페이먼츠 결제창에서 결제 수단 선택 및 결제 진행"

5. **결제 완료 페이지**
   - 스크린샷: https://baroolim.com/payment/success
   - 설명: "결제 완료 확인 페이지"

6. **서비스 이용 페이지**
   - 스크린샷: https://baroolim.com/studio (콘텐츠 생성 페이지)
   - 설명: "결제 후 서비스 이용 가능"

**PPT 제작 방법:**

1. 각 단계별로 스크린샷을 찍습니다.
2. PowerPoint에서 새 프레젠테이션을 만듭니다.
3. 각 슬라이드에 스크린샷과 설명을 추가합니다.
4. 파일명: `바로올림_결제경로파일_YYYYMMDD.pptx`
5. 토스페이먼츠에 이메일로 첨부하여 제출합니다.

**참고:**
- 스크린샷은 실제 결제 과정을 보여주는 것이 좋습니다 (테스트 결제 가능).
- 각 슬라이드에 단계 번호와 설명을 명확히 기재합니다.
- 결제창 화면은 개인정보가 보이지 않도록 모자이크 처리합니다.
